# 介绍

通常用于小程序接入阿里图标后在页面中使用，使用前请将/assets/styles/iconfont.scss 中内容替换成自己的阿里图标地址即。

### Props

| 参数  | 说明     | 类型             | 默认值 |
| ----- | -------- | ---------------- | ------ |
| size  | 大小     | number ｜ string | 28rpx  |
| icon  | 阿里图标 | string           | -      |
| color | 颜色     | string           | -      |

### 外部样式类

| 类名                     | 说明           |
| ------------------------ | -------------- |
| external-custom-iconfont | 阿里图标样式类 |
